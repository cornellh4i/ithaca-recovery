## Getting Started


> Folder structure 

    .
    ├── frontend      # Next.js client
    └── README.md

### Prerequisites
* Docker

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:hahnandrew/recovery.git
   ```
2. Start the client
   ```sh
   cd frontend
   yarn install
   yarn run dev
   ```

> Note: See individual project files for more information on how to build and deploy the project.
